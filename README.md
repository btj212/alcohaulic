# ALCOHAULIC

Long-haul trucker survival roguelike. Stay drunk enough to function, sober enough to drive, employed enough to eat.

Every other game punishes you for drinking. This one punishes you for sobering up.

## Play

```bash
npm install
npm run dev
```

Production build:

```bash
npm run build
npm run preview
```

Tests (pure meters):

```bash
npm test
```

## Controls

| Key | Action |
|---|---|
| A / D or ← → | Steer |
| W / S or ↑ ↓ | Boost / brake (cruise is automatic) |
| 1 | Beer |
| 2 | Liquor |
| 3 | Coffee |
| P | Pull-off (shop / sleep) |

## Week-sprint scope

Browser vertical slice: highway drive + BAC pocket + drunk-vision + death cards + menu pull-off. No cops, no 3D hubs, no Steam wrapper yet.

# ALCOHAULIC

Long-haul trucker survival roguelike. Stay drunk enough to function, sober enough to drive, employed enough to eat.

Every other game punishes you for drinking. This one punishes you for sobering up.

## Play

```bash
npm install
npm run dev
```

Production build:

```bash
npm run build
npm run preview
```

Tests (pure meters):

```bash
npm test
```

## Controls

| Key | Action |
|---|---|
| A / D or ← → / mouse | Steer (subtle; mouse = fine adjust) |
| W / S or ↑ ↓ | Boost / brake (cruise is automatic) |
| 1 | Beer |
| 2 | Liquor |
| 3 | Pills |
| P | Pull-off (shop / sleep) |

You play **Earl "Smokey" Jackson** — 22 years over-the-road, zero sober sunrises. Marlene runs dispatch, Preacher haunts channel 19, and Lucy keeps the only lights on for sixty miles.

First stop: **Lucky Lucy’s** strip club ~5 minutes in (mile ~4.5). Neon lights up on the right as you approach; dispatch cues the exit.
